# ElmenusTask
Create a simple app that connects to elmenus mock API to display a list of tags and menu items for each tag




### Prerequisites
- macOS 10.14.5+
- Xcode 11.3+
- Cocoapods 1.8.4+

### Installing
` pod install `

open ElmenusTask.xcworkspace in Xcode
